# golang-wxwork-sdk

#### 介绍
企业微信，企业方服务API提供，企业自行开发扩展，GO语言服务端+docker自动化构建

#### 软件架构
1、GOLANG

2、Swagger generater

3、OAuth2.0

4、Docker File

5、Jenkins with docker-compose.yml


#### 安装教程

将企业的corpid,secret配置到conf/app.conf
该API采用了OAuth2.0安全协议，访问前可以通过以下方式获取令牌
client_id:lulouis
client_secret:123456
该认证信息可连接您自身的员工数据库进行认证
